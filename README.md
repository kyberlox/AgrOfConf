## Модуль описания продуктовой линейки

Данный модуль описывает продуктовую линейку. Собирается из пакетов:

### Продукт (Product)

Сущность для описания продуктовой линейки, которая включает в себя атрибуты:
- id
- name
- description
- manufacturer
- image

Методы:
- add (заполнить и создать)
- add_by_table (создать с помощью таблицы Excel)
- update (обновить)
- delete (удалить)
- add_parameter (добавить параметр)
- delete_parameter (удалить параметр)
- get (получить)
- get_all (получить все)
- get_parameters (получить параметры продукта)

### Тип параметра (parameter_type)
Атрибуты:
- id
- name
- description

Параметры бывают нескольких типов:
- строки
- числа
- интервал чисел

Методы:
- add (заполнить и создать)
- update (обновить)
- delete (удалить)
- get (получить)
- get_all (получить все)

### Параметр (parameters)

Сущность для описания параметров продуктовой линейки, которая включает в себя атрибуты:
- id
- name
- description
- image
- product_id
- body:
    - parameter_type_id
    - value

Методы:
- add (заполнить и создать)
- update (обновить)
- delete (удалить)
- get (получить)

### Тип связи (specification_type)

Атрибуты:
- id
- name
- description

Параметры бывают нескольких типов:
- один к одному $$ a_{1} => b_{1} $$
- один ко многим $$ a_{1} => (b_{1}, b_{2}, b_{3}) $$
- функция $$ a_{1} => b_{1} = f(a_{1}) $$
- функция нескольких параметров $$ a_{1}, c_{1} => b_{1} = f(a_{1}, c_{1}) $$

Методы:
- add (заполнить и создать)
- update (обновить)
- delete (удалить)
- get (получить)
- get_all (получить все)

### Связь (Specification)

Сущность для описания связей параметров продуктовой линейки, которая включает в себя атрибуты:
- id
- name
- description
- parameter_id
- body:
  - value
  - specification_type_id

Методы:
- add (заполнить и создать)
- update (обновить)
- delete (удалить)
- get (получить)
